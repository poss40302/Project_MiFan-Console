# 🛸 MiFan-Console 使用者手冊 (User Manual)

歡迎使用 **MiFan-Console**！這是一枚為指揮官量身打造的桌上型「戰術科技 HUD」，讓您無需打開手機，即可透過電腦螢幕上的懸浮面板精準操控您的智米風扇。

---

## 1. 📂 準備工作：獲取設備憑證

在開始操作前，您需要取得風扇的 **IP 地址** 與 **Token**。
> [!IMPORTANT]
> **環境要求**：請確保您的 **電腦與風扇連接在同一個區域網路 (Wi-Fi/路由器)**。本軟體透過區域網路 UDP 協定與風扇通訊，無法跨網段或透過雲端遙控。

### 建議工具：Xiaomi Cloud Tokens Extractor
- **推薦原因**：這是目前最簡單、且不需要 Root 手機的方法。
- **操作步驟**：
  1. 從 GitHub 下載並執行 [Xiaomi Cloud Tokens Extractor](https://github.com/PiotrMachowski/Xiaomi-cloud-tokens-extractor)。
  2. 輸入您的小米帳號與密碼。
  3. 工具會列出您帳號下所有的設備，請找到您的風扇並複製 `IP` 與 `Token`。

---

## 2. ⚙️ 初始化設定

1. 執行 `MiFan-Console.exe`。
2. **右鍵點擊** 懸浮窗開啟選單。
3. 選擇 **「通訊設定」**。
4. 在彈出的視窗中輸入您的風扇 **IP** 與 **Token**。
5. 點擊 **「儲存」**，程式將自動重啟通訊模組。

---

## 3. 🎨 視覺語言：圖示狀態解說

面板中心圓鈕的圖示代表了當前的連線與運作狀態：

| 圖示表現 | 代表意義 | 說明 |
| :--- | :--- | :--- |
| **🌀 藍色風扇旋轉** | **運作中 (Power On)** | 設備連線正常，且風扇正在吹送涼風。 |
| **🔘 灰色停止風扇** | **待機中 (Power Off)** | 設備連線正常，但風扇處於關閉狀態。 |
| **📡 Wifi 異常圖示(Wi-Fi符號+驚嘆號)** | **本機網路異常** | 指揮官的電腦 Wifi 未開啟，或無法連接至預設閘道。 |
| **⚠️ 設備異常圖示(灰色風扇+驚嘆號)** | **通訊超時** | 本機網路正常，但無法連接至風扇。請檢查風扇電源或 IP 設定。 |

## 4. 🖱 互動操控秘笈

本工具設計為「重度感應式操作」，請善用以下控項：

- **左鍵點擊 (風扇圖示圓鈕)**：切換電源開關。
- **長按左鍵並拖曳**：自由移動面板位置（視窗會自動記憶最後停留位置）。
- **切換鈕 (Pill-Switch)**：一鍵切換「直吹風」與「自然風」。
- **風量/級數調整**：
    - **直吹風模式**：使用 **滑鼠滾輪** 或進度條進行 1%-100% 的精確微調。
    - **自然風模式**：點擊下方 **四段式按鈕** 直接切換強度等級。
- **擺頭開關 (Oscillate)**：控制風扇左右擺動。
- **右鍵選單**：
  - **開機自動啟動**：讓 HUD 與您的 Windows 同步啟動。
  - **通訊設定**：修改 IP/Token。
  - **離開系統**：完全退出並關閉本程式。

## 5. 🛠 常見問題排除 (FAQ)

- **Q: 為什麼我的面板會自動「縮回去」？**
  - A: 當診斷系統偵測到網路異常或設備斷電時，會自動鎖定控制項並縮回 compact 模式以節省空間與無效通訊。連線恢復後若風扇為開啟狀態，面板會自動彈回展開。
- **Q: 面板消失在螢幕邊界外了怎麼辦？**
  - A: 程式具備「座標自動校正」功能。若您拔除螢幕導致視窗越位，下次啟動時它會自動跳回主螢幕中央。
- **Q: 會被防毒軟體誤判嗎？**
  - A: 本軟體使用 Nuitka 進行原生編譯，避開了常見的 PyInstaller 誤報問題，提供更安全的使用體驗。
